Lissajou curves

The Lissajou curves is the result of displaying graphically the result of these formulas:

x(t) = Asin(⍵t)

y(t) = Bsin(Ωt + δ)

Obtaining values for x and y for every value of t in a single frame. ⍵ represents the horizontal frequency and Ω represents the vertical one.
